CSM Reachability Logical Data Model
In this section we will explore the logical data model used by CSM Reachability Service. The data structures on the data model are all dynamic configurations and supported by DPS setting management APIs.
Updates in Dynamic Configuration (after an optional approval process) are applied on a (near) real-time basis during the processing of payments in IPF.
