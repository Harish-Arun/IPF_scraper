IPF Licensing
Reporting Usage Data to IPF Support
If IPF support requires a report of used IPF components these will be available to view at the actuator endpoint
actuator/ipflicense
in JSON format. The contents of the data is encrypted and only IPF support teams can view the plaintext data. The data in this page can be sent to IPF support teams.
